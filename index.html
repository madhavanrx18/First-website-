Welcome to my website!

I would like to give a introduction about the pages that are linked in the navbar.

1) About page : The "About" page is an integral part of our website that provides essential information about our organization and its mission. It aims to offer insights into the core values, achievements, and contributions of our institution. Additionally, this page presents valuable details about the prominent figures associated with our organization and highlights the key features of our educational programs. Through interactive buttons(activated only through mouse-events), users can explore specific events and notable aspects related to our establishment.

2) CGPA Calculator :This page features a dynamic CGPA (Cumulative Grade Point Average) calculator that allows users to input their course details such as course code, course name, credits, and the corresponding grade. Users can then calculate their CGPA by clicking the "Calculate CGPA" button, which displays the total grade points, total credits, and the calculated SGPA (Semester Grade Point Average) based on the provided data. Additionally, the page includes a table for mapping various grade points based on specific grades.

3) Login page : Various tools are used to ensure security agaisnt bot attacks and safe user experience
Captcha Implementation: Captchas are used to distinguish between human users and automated bots. By generating a random string of characters and displaying it as a challenge to the user, the page ensures that only a human can successfully log in.
Hidden Form Submission Check: The page includes a hidden form submission check that helps detect if any automated bot attempts to submit data. If the hidden form fields are filled, the system alerts the user about a potential robot activity.

4) Register PageThe form includes various input fields, radio buttons, checkboxes, and dropdown menus to gather specific information from the users. The form collects essential details such as the user's name, email, contact number, current CGPA, preferred department, fee slab, language proficiency, and the preferred method of stay. The user is required to fill in all the necessary fields to successfully submit the registration form. Additionally, the form has validation checks to ensure that all required fields are completed and that at least one language option is selected. Upon clicking the "Submit" button, the user's inputs are validated, and if all required fields are appropriately filled and at least one language is selected, the user receives an alert confirming the successful registration.
